Screenshot Extension Test Page

How to test the extension:

  1. Load the extension in Chrome (chrome://extensions/)
  2. Enable Developer mode and click "Load unpacked"
  3. Select the hunt-melodic-duck/build/chrome-mv3-dev folder
  4. Use the keyboard shortcut Ctrl+Shift+S (or Cmd+Shift+S on Mac)
  5. Or click the extension icon and use the "Capture Screenshot" button
  6. The screenshot should be copied to your clipboard automatically
